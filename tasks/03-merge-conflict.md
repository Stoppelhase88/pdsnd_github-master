# Aufgabe 3: Merge-Konflikt gezielt auslösen & lösen

**Ziel:** Konflikte verstehen und sauber auflösen.

1. Erzeuge **parallel** einen zweiten Branch `feature/readme-polish` und ändere denselben Absatz im `README.md` wie in deinem ersten Branch.
2. Merge zuerst Branch A, dann Branch B in `main` per PR, löse den Konflikt im GitHub-UI oder lokal.
3. Schreibe eine **Merge-Commit-Message**, die den Konflikt und die Lösung dokumentiert.
